Leadership Signal Read-out
⚡ What the data says — 4 signals for your decision
The product is not delivering its core value moment at scale. Only 4–6% of users who install complete a book summary on Day 0. The main bottleneck is between "Opens Content" (13–31%) and "Completes Summary" — meaning users start but abandon before finishing. Fixing this single drop-off unlocks everything downstream.
Retention is critically below category benchmarks and declining. D1 retention of 5–10% is 4–7× below the 35% benchmark for Book Summary apps. D7 retention at 0–5% means there is virtually no habit formation. The latest concept test shows D7 = 0% — the trend is moving in the wrong direction.
A creative-quality tension is attracting wrong-fit users at scale. "Sex Skill" creative delivers the lowest CPI ($1.5) but the worst D1 retention (3%). "Image-04" delivers the best D1 (19%) but costs $5.00/install with only 32 installs tested. Until a creative combines low CPI AND high retention, UA spend is building a leaky bucket.
Recommended next action: Do not scale UA budget. Invest next sprint in fixing first-session onboarding (target: get Complete First Summary rate from 6% → 30%+). Run a dedicated product test on content discoverability and summary length. Only return to UA scaling once D1 retention exceeds 20% consistently across 2 rounds.

Looker Studio Setup — Beginner Step-by-Step
Goal: Recreate this dashboard in Looker Studio for live team sharing
Three CSV files are included in your folder: BookWise_Rounds_Data.csv (trends), BookWise_Funnel_Data.csv (funnel), BookWise_Creatives_Data.csv (creatives). Upload all three to Google Sheets first, then follow these steps.
  1. 1

    Upload CSVs to Google Sheets

    Open Google Sheets → File → Import → Upload. Upload BookWise_Rounds_Data.csv first. Name the tab Rounds. Repeat for Funnel (tab: Funnel) and Creatives (tab: Creatives) — either as separate sheets in the same file or three different files. Keep the URL handy.

  2. 2

    Create a Looker Studio Report

    Go to lookerstudio.google.com → Click Create → Report. Click Add data → Google Sheets → select your BookWise sheet → choose the Rounds tab → Add.

  3. 3

    Build the PMF Scorecard Row (4 scorecards)

    Insert → Scorecard (repeat 4×).
    Card 1: Metric = complete_rate_pct · Label = "Onboarding Completion %"
    Card 2: Metric = d1_retention_pct · Label = "D1 Retention %"
    Card 3: Metric = tspu_mins · Label = "Avg Time Spent D0 (min)"
    Card 4: Metric = d7_retention_pct · Label = "D7 Retention %"

  4. 4

    Add Retention Trend Chart

    Insert → Line chart. Dimension = round. Add 3 metrics: d1_retention_pct, d3_retention_pct, d7_retention_pct. Style → Reference lines → Add constant line at 35 (D1 benchmark, color: green, dashed).

  5. 5

    Add CPI & TSPU Trend Charts

    Insert → Bar chart. Dimension = round. Metric = cpi_usd. Duplicate it for TSPU: change metric to tspu_mins. Tip: in Style → Bar color → set conditional color rules (high CPI = red, low = green).

  6. 6

    Add Funnel Chart (new data source)

    Click Resource → Add a data source → Google Sheets → Funnel tab. Insert → Bar chart. Dimension = stage. Metric = pct_of_installs. Filter control: add a Filter on round so leadership can toggle R6 vs. R7. Sort: stage_order ascending.

  7. 7

    Add Creative Performance Table

    Add Creatives as a third data source (same method as step 6). Insert → Table. Columns: creative, spend_usd, installs, cpi_usd, d1_pct, d3_pct, notes. Style → enable Data bars for cpi_usd and d1_pct columns. Sort default: d1_pct descending.

  8. 8

    Add Round Filter Control

    Insert → Filter control → set field = round. This lets leadership click between rounds on any chart. Place it top-right of the report. Also add a Date range control connected to the start_date field.

  9. 9

    Share with Leadership

    Top right → Share → Manage access → Anyone with link → Viewer. Copy link and paste into Slack/email. Anyone can view without a Google account. To update: just paste new round data rows into the Google Sheet — Looker auto-refreshes within minutes.
